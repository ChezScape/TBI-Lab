# Tower Battle Intel v4.11a5 Command Deck Dedup

## Purpose

The Command Deck top panel had duplicate controls that already exist in the Battle Report Input console below it:

- Save Report
- Clear Input
- Clear Runs

That made the desktop command flow confusing.

## Change

The top Command Deck now uses shortcut actions only:

- Open History
- Import History
- Export History
- Open Debug

The Battle Report Input console below keeps the real report input controls:

- Save Report
- Build select
- Clear Input
- Clear Runs

## Scope

Desktop-only polish. Mobile remains untouched for the later v4.11b pass.
